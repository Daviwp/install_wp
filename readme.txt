== Description ==

This script is in its absolute location, it serves for you to install local wordpress on your server, it makes it easier because it does everything for you, downloads the latest theme from the official WordPress repository, installs the user bank and password you define without changing a line of code. It is possible to improve your way. It makes it easy to quickly upload a local wordpress. It was made to work on OSX - MAC, and Linux operating systems, but it can also modify parameters and directories for Windows systems.
